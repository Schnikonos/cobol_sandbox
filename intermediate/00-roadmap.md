# COBOL Intermediate Roadmap

This track builds on `beginner/` and focuses on practical batch-programming patterns.

## Study order

1. `01-numeric-precision.md` + `01-numeric-precision.cob`
2. `02-string-processing.md` + `02-string-processing.cob`
3. `03-tables-and-search.md` + `03-tables-and-search.cob`
4. `04-modular-flow.md` + `04-modular-flow.cob`
5. `05-validation-patterns.md` + `05-validation-patterns.cob`
6. `06-file-status-and-robust-io.md` + `06-file-status-and-robust-io.cob`
7. `07-sort-and-merge-basics.md` + `07-sort-and-merge-basics.cob`
8. `08-report-formatting.md` + `08-report-formatting.cob`
9. `09-program-calls.md` + `09-program-calls-main.cob` + `09-program-calls-sub.cob`
10. `exercises.md`

## Goal

At the end of this track, you should be able to write small robust batch programs with validation, table lookup, file/report handling, and subprogram reuse via `CALL`.
