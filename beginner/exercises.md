# COBOL Exercises (Beginner)

Try these without looking at solutions first.

## Exercise 1: Personal card

Create `ex01-personal-card.cob` that:

- Asks for first name
- Asks for age
- Displays: `Name: <name> | Age: <age>`

## Exercise 2: Pass/fail checker

Create `ex02-pass-fail.cob` that:

- Reads a score (0-100)
- Displays `PASS` if score >= 50 else `FAIL`

## Exercise 3: Counter + sum

Create `ex03-counter-sum.cob` that:

- Prints numbers 1 to 5
- Calculates and displays total sum

Expected sum: `15`.

## Exercise 4: Grade with EVALUATE

Create `ex04-grade.cob` that maps score to grade:

- A: 90+
- B: 80-89
- C: 70-79
- D: 60-69
- F: below 60

## Exercise 5: File practice

Create `ex05-file-write-read.cob` that:

- Writes 3 product lines to a file
- Reads and displays each line
