# Lesson 1: Program structure + DISPLAY

A minimal COBOL program has:

- `IDENTIFICATION DIVISION`
- `PROGRAM-ID`
- `PROCEDURE DIVISION`

Example flow:

1. Start program.
2. Execute statements in `PROCEDURE DIVISION` from top to bottom.
3. `STOP RUN` ends execution.

## Key statement

- `DISPLAY` prints text (or variables) to the screen.

## Try this

- Change the displayed text.
- Add a second `DISPLAY` line.
- Remove `STOP RUN` and see if compiler warns.
